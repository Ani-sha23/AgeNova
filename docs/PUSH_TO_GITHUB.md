# Push This Repo to GitHub

Target repository:

```text
https://github.com/Ani-sha23/AgeNova
```

## First Push

From the `AgeNova` folder:

```bash
git init
git branch -M main
git add .
git commit -m "Initial AgeNova project"
git remote add origin https://github.com/Ani-sha23/AgeNova.git
git push -u origin main
```

If GitHub asks for credentials, use a GitHub personal access token instead of a password.

## After Editing

```bash
git add .
git commit -m "Update AgeNova"
git push
```
